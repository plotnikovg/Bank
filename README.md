# 🏦 Банковское веб-приложение
 
Учебное full-stack банковское приложение, построенное с помощью **ASP.NET Core** и применением **Чистой архитектуры**, реализующее банковские операции с фокусом на безопасности.

---

## Features
 
- **Аутентификация и авторизация** — построены на JWT
- **Управление счетами** — cоздание банковских счетов
- **Переводы** — денежные переводы между счетами
- **Безопасность** — разработка с учётом рекомендаций OWASP Top 10

---

## Технологический стек
 
| Слой | Технология |
|---|---|
| Backend | ASP.NET Core |
| ORM | Entity Framework Core |
| Архитектура | CQRS + MediatR |
| Аутентификация | JWT Bearer Tokens |
| База данных | MS SQL |
| Frontend | HTML / CSS |

---

## Архитектура
 
Проект следует принципам **Clean Architecture** и разделён на четыре независимых слоя:
 
```
Bank/
├── Bank.Domain/          # Сущности, доменные правила, интерфейсы
├── Bank.Application/     # Сценарии использования, CQRS-обработчики
├── Bank.Infrastructure/  # EF Core, репозитории, миграции БД
└── Bank.API/             # REST API на ASP.NET Core, контроллеры

─── Bank.WebClient/       # Веб-клиент (HTML/CSS)
```

---

## Безопасность
 
Безопасность является приоритетом в данном проекте. Реализована защита от уязвимостей из списка OWASP Top 10, включая:
 
- Валидацию входных данных
- JWT-токены с ограниченным сроком действия и корректной обработкой claims
- Защиту от SQL-инъекций через параметризованные запросы EF Core
- Принудительное использование HTTPS в production-конфигурации
