# 🏦 Банковское веб-приложение
 
Учебное full-stack банковское приложение, построенное с помощью **ASP.NET Core** и применением **Чистой архитектуры**, реализующее банковские операции с фокусом на безопасности.

---

## Features
 
- **Аутентификация и авторизация** — построены на JWT
- **Управление счетами** — cоздание банковских счетов
- **Переводы** — денежные переводы между счетами
- **Безопасность** — разработка с учётом рекомендаций OWASP Top 10

---

## Технологический стек
 
| Слой | Технология |
|---|---|
| Backend | ASP.NET Core |
| ORM | Entity Framework Core |
| Архитектура | CQRS + MediatR |
| Аутентификация | JWT Bearer Tokens |
| База данных | MS SQL |
| Frontend | HTML / CSS |
